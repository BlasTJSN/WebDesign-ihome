1/生成图片验证码
请求方法:GET
请求url:/api/v1.0/imagecode/<image_code_id>
请求参数:除UUID外无参

返回结果:
正确情况下:
返回图片本身:不是json数据
resp = make_response(image)
resp.headers['Content-Type'] = 'image/jpg'

错误情况下:
return jsonify(errno=RET.DBERR,errmsg='存储图片验证码失败')

imagecode_uuid = 'abcd'

2/发送短信
请求方法:GET
请求url:/api/v1.0/smscode/?<mobile>&text='abcd'&id='2asdf234'
请求参数:
参数名         是否必须        参数描述
mobile          是           用户输入的手机号
text            是           用户输入的图片验证码的内容
id              是           图片验证码的编号
返回结果:
正确情况下:
return jsonify(errno=RET.OK,errmsg='OK')

错误情况下:
return jsonify(errno=RET.DATAERR,errmsg='图片验证码错误')

11/发布新房源
请求方法:POST
请求url:/api/v1.0/houses
请求参数:
参数名         是否必须        参数描述
title           是           房屋标题
price           是           房屋价格
area_id         是           房屋城区
address         是           房屋详细地址
room_count      是           房屋数目
acreage         是           房屋面积
unit            是           房屋单元(几室几厅)
capacity        是           人数上限
beds            是           卧床配置(单人床/双人床)
deposit         是           押金
min_days        是           最小入住天数
max_days        是           最大入住天数
facility        否           配套设施

返回结果:
正确情况下:
return jsonify(errno=RET.OK,errmsg='OK',data={'house_id':house.id})

错误情况下:
前端是以元为单位,数据库中为了确保数据的准备,需要由元转成分
price = int(float(price*100))
return jsonify(errno=RET.DATAERR,errmsg='价格数据错误')

12/上传房屋图片:
request.args(url中查询字符串,键值对)---request.form(表单中)---request.data(xml)---request.get_json(请求体中的json数据)

请求方法:POST
请求url:/api/v1.0/houses/<int:house_id>/images
请求参数:
参数名         是否必须        参数描述
house_id        是           房屋的id
house_image     是           房屋图片

返回结果:
正确情况下:
image_url = 七牛外链域名 + 七牛返回的图片名称
return jsonify(errno=RET.OK,errmsg='OK',data={'url':image_url})
错误情况下:
return jsonify(errno=RET.THIRDERR,errmsg='上传图片失败')
